# 社員紹介サイト
- ログイン機能(メールアドレスとパスワード)
- ログインしたユーザは自身のプロフィールを編集可能
- プロフィールには複数の職歴を含むことが可能
- ログインしたユーザは社員のプロフィール一覧を閲覧可能
- ユーザーには、admin権限を持ったユーザーと、そうでないユーザーがいる
- admin権限を持ったユーザーは、ユーザーの追加・削除が可能
